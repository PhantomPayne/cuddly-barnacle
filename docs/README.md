## About Barnacle

Barnacle is a new programming language that compiles to WebAssembly Components, featuring:

- **Algebraic effects** as the universal abstraction for side effects
- **Wasm Component Model** integration where effects map to WIT interfaces
- **Compile-time metaprogramming** via sandboxed Wasm components
- **First-class LSP** built on Salsa incremental computation
- **Complete dead code analysis** enabled by explicit entry points and effect tracking
